## Prerequisites
This repository is tested ...
```
- Windows 10

- Unreal Engine 4.25
- UE4 tcp socket plugin (https://unrealengine.com/marketplace/en-US/product/tcp-socket-plugin)

- Visual Studio 2019

- Python 3.6
- Pytorch 1.7.1

```
